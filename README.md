# Colossal cave adventure

This is a faithful port of the “Adventure” game to a Mycroft Skill from the original 1977 FORTRAN code by Crowther and Woods

- who made cave adventure
- play cave adventure
- enter building
- get food
- save
- quit
- resume cave adventure
